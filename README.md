# Scrutiny Server in Rust

This is a Rust implementation of the Scrutiny Server, which is a tool for monitoring and analyzing data from various sources. The server can be used to collect data from different sensors and devices, and provides a web interface for visualizing and analyzing the collected data.

Run the server example application sine_source:

cargo run --example sine_source


Run the Scrutiny GUI:

scrutiny gui    --auto-connect
